# Wicked Wight CLI

Generate random, fun passwords from the command line.

## Options

```
  -m, --mixed         Use mix of words and numbers (default: false)
  -n, --num <number>  Number of phrases to create (default: "5")
  -l, --len <number>  Length of each phrase (default: "5")
```
Examples:

```
wicked-pw --len 4
```
