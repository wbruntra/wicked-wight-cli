# Wicked Wight CLI

Generate random, fun passwords from the command line.

## Options

```
-n, --num   Number of phrases [number]
-l, --len   Length of phrases [number]
-m, --mixed Generate mixed phrases (words and numbers)
```
Examples:

```
wicked-pw --len 4
```
